🚀 COMO USAR
────────────────────────────────────────────────────────────────────────────

OPÇÃO 1: Abrir Localmente (Mais Rápido)
─────────────────────────────────────────
1. Descompacte todos os arquivos em uma pasta
2. Clique duas vezes em "index.html"
3. A página abrirá no seu navegador padrão
4. Pronto! Sem necessidade de servidor

OPÇÃO 2: Servidor Local (Python)
─────────────────────────────────
1. Abra o terminal/cmd na pasta do projeto
2. Execute: python -m http.server 8000
3. Acesse: http://localhost:8000
4. Pressione Ctrl+C para parar

OPÇÃO 3: Servidor Local (Node.js)
──────────────────────────────────
1. Instale: npm install -g http-server
2. Abra o terminal/cmd na pasta do projeto
3. Execute: http-server
4. Acesse: http://localhost:8080

OPÇÃO 4: Upload para Servidor Web
──────────────────────────────────
1. Faça upload de todos os arquivos via FTP/SFTP
2. Coloque na raiz do seu domínio
3. Acesse via seu domínio
4. Pronto!

🎨 PERSONALIZAÇÃO
────────────────────────────────────────────────────────────────────────────

ALTERAR CORES:
1. Abra "assets/css/styles.css"
2. Procure por ":root {"
3. Modifique as variáveis:
   --mls-gold: #D4AF37;      (Dourado)
   --mls-navy: #001F3F;      (Azul Marinho)
   --mls-cyan: #00D4FF;      (Ciano)

ALTERAR TEXTOS:
1. Abra "index.html"
2. Procure pelo texto que deseja mudar
3. Edite conforme necessário

ALTERAR LOGO:
1. Substitua "logo_mls_circuito_dourado.png" por sua imagem
2. Mantenha o mesmo nome ou atualize em index.html
